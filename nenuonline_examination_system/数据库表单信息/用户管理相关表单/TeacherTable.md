# 教师表

## 表名

teacher

## 内容

| 序号 | 列名          | 说明     | 数据类型 | 长度 | 主键 | 是否为空 |
| ---- | ------------- | -------- | -------- | ---- | ---- | -------- |
| 1    | Uid           | 主键     | INT      |      | 是   | 否       |
| 2    | TeacherID     | 教师编号 | INT      |      | 否   | 否       |
| 3    | TeacherName   | 教师名   | VARCHAR  |      | 否   | 否       |
| 4    | TeacherPwd    | 用户密码 | VARCHAR  |      | 否   | 否       |
| 5    | Tel           | 电话     | INT      |      | 否   | 是       |
| 6    | TeacherAdr    | 通讯地址 | VARCHAR  |      | 否   | 是       |
| 7    | Introduction  | 简介     | TEXT     |      | 否   | 是       |
| 8    | TeacherSchool | 所属学校 | VARCHAR  |      | 否   | 否       |
